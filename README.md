# Movie-Recommender-System-with-sentiment-analysis

## How to run the project?

1. Clone or download this repository to your local machine.
2. Install all the libraries mentioned in the [requirements.txt]
3. Get your API key from https://www.themoviedb.org/. 
4. Replace your_api_key in **both** the places of `static/recommend.js` file and preprocessing 3.ipynb and hit save.
5. Open your terminal/command prompt from your project directory and run the file `main.py` by executing the command `python main.py`.
6. Go to your browser and type `http://127.0.0.1:5000/` in the address bar.
7. Hurray! That's it.
